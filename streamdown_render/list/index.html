List rendering.

Renders markdown lists with:

  • Bullet points (•, ◦, ▪)
  • Ordered numbers with configurable style
  • Nested indentation
  • Proper text wrapping for long items
  • Inline formatting (bold, italic, strikethrough, etc.)
