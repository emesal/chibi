Module model_info

Expand description

Model metadata retrieval and formatting.

Two output formats:

  • TOML (format_model_toml): For copy-pasting into config.toml (CLI -m/-M).
  • JSON (format_model_json): Structured output for the model_info tool.

Metadata is fetched via fetch_metadata, which delegates to ratatoskr’s ModelGateway::fetch_model_metadata() (registry → cache → network).
