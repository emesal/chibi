Debug logging utilities for API requests and responses.

These utilities log API interactions to JSONL files in the specified context’s directory when debug logging is enabled.

log_to_jsonl
Log data to a JSONL file in the specified context’s directory if debug logging is enabled. The required_key specifies which DebugKey enables this log. DebugKey::All always matches.
