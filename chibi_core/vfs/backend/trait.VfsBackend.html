Expand description

Storage backend for the virtual file system.

All methods receive validated VfsPath values. Backends translate these to their native addressing (file paths, database keys, XMPP nodes, etc.).

All methods are async (returning boxed futures) to accommodate network-backed implementations and dyn-compatibility.
