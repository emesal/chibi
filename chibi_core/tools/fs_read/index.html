Module fs_read 

Source
Expand description

fs_read tools: read-only access to OS and VFS paths. file_head, file_tail, file_lines, file_grep, dir_list, glob_files, grep_files. No PreFileRead/PreFileWrite hooks are fired here; gating is in the dispatcher.
