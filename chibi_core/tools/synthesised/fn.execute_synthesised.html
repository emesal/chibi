Execute a synthesised tool by calling its bound execute procedure.

Converts JSON args to a scheme alist, resolves the exec_binding in the session, and calls it with stdout/stderr capture. Returns structured output: "result: <value>\nstdout: <output>\nstderr: <output>".

Sets BRIDGE_CALL_CTX (with the per-call registry) via CallContextGuard before calling into scheme so that call-tool can access the runtime context.

%context-name% is injected via set! on the top-level binding from HARNESS_PREAMBLE. Concurrent calls to different TeinSessions are safe because each session has its own BRIDGE_CALL_CTX slot (keyed by worker_thread_id). Calls to the same session are serialised by ThreadLocalContext’s internal mutex, so no interleaving can occur.
