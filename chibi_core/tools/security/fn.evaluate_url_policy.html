pub fn evaluate_url_policy(
    url: &str,
    safety: &UrlSafety,
    policy: &UrlPolicy,
) -> UrlAction
Expand description

Evaluate a URL against a policy. Returns the action to take.

url is the original URL string (for glob matching). safety is the result of classify_url() (for category matching).

For glob matching, the URL is canonicalized (lowercased, parsed and re-serialized) to prevent bypasses via case or encoding tricks.
