Module vfs_tools

VFS tools for virtual filesystem operations.

Thin async wrappers around Vfs methods, exposed as LLM-callable tools. Each tool parses vfs:// URIs from JSON args and delegates to the appropriate Vfs method, returning human-readable result strings.
