# Example Skill

This is a minimal example skill to demonstrate the Agent Skills system.

When this skill is invoked, you should:

1. Greet the user
2. Explain that you're operating in "example mode"
3. Offer to help with a demonstration

This skill has no tool restrictions, so all tools remain available.

## Usage

To invoke this skill, call the `skill_example-skill` tool.
